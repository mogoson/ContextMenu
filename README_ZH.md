# MGS-ContextMenu
- [English Manual](./README.md)

## 概述
- Unity 制作UGUI上下文菜单 插件包。

## 需求
- Unity 场景中，鼠标右键点击目标物体时弹出上下文菜单，点击菜单项时对目标物体执行相应的操作。

## 环境
- Unity 5.3 或更高版本。
- .Net Framework 3.0 或更高版本。

## 实现
- ContextMenuUI：控制上下文菜单UI。

## 案例
- “MGS-ContextMenu\Scenes”目录下存放上述功能的演示案例，供读者参考。
